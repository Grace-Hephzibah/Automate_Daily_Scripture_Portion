# Automate_Daily_Scripture_Portion
LDH Build 2022 : Day 1 - Automate a Daily Task

## Aim
This code automates the daily part of me sending the Scripture portion individually to people who have joined with me in this year's (2022) challenge

## Working 
- The project consists of various files in the name of all the months which in it contains the portion of the day. 
- There is also a code to take out the particular day's portion
- Then the code works to send that portion to all the people found in the file numbers 
- And Voila! That works !!!
